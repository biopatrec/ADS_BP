void LSM6DS3AcquireSample ( imuStruct *  sample)

This function reads the a new sample from both accelerometer and gyroscope.

Parameters
[out]*samplepoints a imuStruct where to store the values
Returns
none.

It works by polling on the INT1 line. The new values are saved on the imuStruct variable passed as an argument to the function.

unsigned char LSM6DS3Init ( void  )

The aim of this function is to initialize the LSM6DS3 and prepare it to acquire samples from both accelerometer and gyroscope.

Parameters
[in]none
Returns
a byte which is 1 if a device was found, 0 otherwise.

The standard configuration is: Accelerometer: all axes enabled at 416 Hz (High performance mode) Gyroscope: all axes enabled at 416 Hz (High performance mode) Device has been tested with a MCU TM4C123 via SPI with the following setup: SSIConfigSetExpClk(SSI2_BASE, clockFreq, SSI_FRF_MOTO_MODE_0, SSI_MODE_MASTER, 8000000, 8); The function returns 1 if the device is found otherwise 0.
