void OBhandReadBuffer ( OBhData *  data)

Function to switch the hand into digital mode.

Parameters
[out]*datapoints the OBhData structure where store data
Returns
none.

The values sent from OB hand every 12 mS are: Frame-start, length, Ele_auf, Ele_zu, Akku, State, Bügel, Pad1, Pad2, Pad3, Strom, CS Frame-start value is: 255. The length is fixed with 10. The other values are in a byte-format (0..255), but only contained values 0..254. If 255 is to send, 254 would be sent. This point has to be considered by external programs.
